Dollar Digger

Dollar Digger was the first 2D game I made using the Unity engine. With this project I learned that solid planning is very important, even when dealing with projects that seem small and/or easy. I started setting up the project and writing code way too early, and as a result, I ran into a lot of problems that could have been solved during a planning phase.

Because of weak planning, Dollar Digger took way too long to complete and could have been much better.
