Voxel Snake

Voxel Snake is the first game I created and published.

After following along with some Youtube tutorials and making some sample projects, I decided to try to make something on my own. As much as I wanted to go straight to making something awesome and original, recreating a simple game that already exists was a much better exercise for a beginner. So I took the basic knowledge I got from some tutorials and spent 3-4 weeks making Voxel Snake.

The game could have used some extra polish and code optimization, but I was too excited to publish it online and forgot those final touch-ups.
